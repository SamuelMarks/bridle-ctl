# Skills

## Purpose

`SKILLS.md` defines the list of AI-enabled capabilities supported out-of-the-box.

1. `CodeTool` Execution via FFI.
2. Git Forge CRUD Operations (Issues, PRs, Comments).
3. Code Search & Batch Mutations via TUI workflows.
4. Smart Upstream PR Synchronization: Native capacity to search for existing personal forks, push code, and create PRs with global limits.
5. Dynamic Database interaction (SQLite/PostgreSQL).
